# ⚡ HackMatch

## 🎯 The Problem We Solve

> *"I wanted to join the hackathon, but everyone already had a team."*

Sound familiar? You're not alone.

Every semester, hundreds of talented students skip hackathons — not because they lack skill, but because they don't have teammates. Most teams form inside tight friend groups, leaving solo students, beginners, and newcomers on the outside looking in.

**HackMatch changes that.** We built the app we wish existed when we were first-year students with big ideas and no one to build with.

---

## ✨ What Makes HackMatch Different

| Feature | What It Does |
|---|---|
| 🔥 **Swipe Matching** | Tinder-style teammate discovery — swipe right on people whose skills complement yours |
| 🤖 **AI Compatibility Score** | Get instant match scores like *"92% Compatible — Strong frontend/backend synergy"* |
| 💬 **Real-Time Chat** | Message your matches, form a group chat, and start coordinating immediately |
| 🏆 **Event-Based Rooms** | Browse real hackathons, join the event, and match with participants going to the same one |
| 👥 **Team Dashboard** | Assign roles, spot skill gaps, and view your team's chemistry score before the event |
| 🌱 **Beginner Mode** | New to tech? Display a beginner badge and get matched with mentors who want to guide you |

## 🚀 How It Works

**It's as easy as 1–2–3–match.**

```
1. Create your profile        →  Add your skills, role preferences, and portfolio
2. Pick your hackathon        →  Browse upcoming events and join the ones you want
3. Swipe on teammates         →  Left = pass  |  Right = interested  |  ⭐ = Super Match
4. It's a Match! 🎉           →  Chat instantly, build your team, and win together
```

No awkward cold DMs. No scrolling through spreadsheets. Just swipe, match, build.

## 🏃 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn
- MongoDB / PostgreSQL instance

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/hackmatch.git
cd hackmatch

# 2. Install frontend dependencies
cd frontend
npm install

# 3. Install backend dependencies
cd ../backend
npm install

# 4. Set up environment variables
cp .env.example .env
# Fill in your DB credentials, Firebase config, etc.

# 5. Run the development servers
# Frontend (http://localhost:3000)
cd frontend && npm run dev

# Backend (http://localhost:5000)
cd backend && npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and start swiping! 🎉
